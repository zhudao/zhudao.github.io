MySQL空间函数:FIND_IN_SET

官网如果字符串 str 位于由 N 个子字符串组成的字符串列表中，则返回 1 到 N 范围内的值。字符串列表是由 以 字符分隔的子字符串组成的字符串。如果第一个参数是常量字符串，而第二个参数是 SET 类型的列，则 FIND_IN_SET函数将优化为使用位算术。如果 str 不在 strlist 中，或者 strlist 是空字符串，则返回 0。如果任一参数为 NULL，则返回 NULL。如果第一个参数包含逗号 （，） 字符，则此函数无法正常工作。
