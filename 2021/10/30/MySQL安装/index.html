MySQL联合索引

1 查看 mysql 的安装路径：

[root@bogon ~]# whereis mysql

mysql: /usr/bin/mysql /usr/lib/mysql /usr/share/mysql /usr/share/man/man1/mysql.1.gz

2. 查看 mysql 的安装包：

[root@bogon ~]# rpm -qa|grep mysql

mysql-community-client-5.6.26-2.el6.i686

mysql-community-release-el6-5.noarch

mysql-community-common-5.6.26-2.el6.i686

mysql-community-libs-5.6.26-2.el6.i686

mysql-community-server-5.6.26-2.el6.i686

[root@bogon ~]#

3. 卸载 mysql

[root@bogon ~]# yum remove mysql

删除 mysql 的数据库文件:删除/var/lib/mysql 目录

4. 安装 mysql

第一步：从 oracle 官方网站下载 linux 系统对应的 mysql 的 yum 源包。

第二步：把 yum 源包上传到 linux，安装。

[root@bogon ~]# yum localinstall mysql-community-release-el6-5.noarch.rpm

[root@bogon ~]# yum install mysql-server

第三步：启动 mysql

[root@bogon ~]# service mysqld start

第四步：给 root 用户设置密码

[root@bogon ~]# /usr/bin/mysqladmin -u root password ‘root’

客户端访问 :

本地访问 - 命令控制台访问

/usr/bin/mysql -u 用户名 -p

第五步：远程连接授权, 命令控制台登录 mysql 服务器后使用.

GRANT ALL PRIVILEGES ON . TO ‘myuser‘@’%’ IDENTIFIED BY ‘mypassword’ WITH

GRANT OPTION;

注意：’myuser’、’mypassword’ 需要替换成实际的用户名和密码。

mysql> GRANT ALL PRIVILEGES ON . TO ‘root‘@’%’ IDENTIFIED BY ‘root’ WITH

GRANT OPTION;

5. Rpm 安装法

如果没有网络环境可以使用参考资料中的 mysql-rpm 文件夹下的 mysql 安装包来安装。

安装包有 4 个文件,有安装顺序.安装文件名称前缀定义的顺序,依次安装.

[root@bogon mysql-rpm]# rpm -ivh mysql-community-*

安装后，启动服务、设置密码、远程授权后既可以使用。
