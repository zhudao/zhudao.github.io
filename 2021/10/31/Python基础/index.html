Python基础

注意： Python大小写敏感

1. 数据类型和变量

  • 整数： 正整数，负整数 0 数中间可以用_ 下划线分割，没差别

  • 浮点数： 小数，运算可能有误差 （ / 除法结果是浮点数 // 地板除，结果永远是整数 ）

  • 字符串： 单引号或双引号 括起来的文本。 转义字符 \ 可以转移特殊字符

  • 布尔值： True False (注意大小写)。 也可以通过布尔运算计算

  • 空值： None表示 空值

  • 变量： 指的是变量名，类型不确定。称之为 动态语言。 对应 “静态语言” 定义时 ，确定下来。

  • 常量：通常用全部大写的变量名表示常量（习惯，实际上 无任何机制保证）
